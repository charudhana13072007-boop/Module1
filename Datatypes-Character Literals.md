# Datatypes-Character Literal in Python

## 🎯 Aim
To write a Python program that prints the characters `'T'` and `'a'` using character literals.

## 🧠 Algorithm
1. Print the character `'T'`.
2. Print the character `'a'`.

## 🧾 Program
~~~
print('T')
print('a')
~~~
## Output
T
a
## Result
The program successfully demonstrates the use of character literals in Python. By enclosing the letters in single quotes (or double quotes), Python recognizes them as string data types and outputs them to the console as specified in the algorithm.
