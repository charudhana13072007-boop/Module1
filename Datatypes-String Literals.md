# Datatypes-Read and Print a String in Python

## 🎯 Aim
To write a Python program to read a string from the user and then print it.

## 🧠 Algorithm
1. Assign a variable named `men_stepped_on_the_moon`.
2. Use `input()` to read a string from the user and store it in the variable.
3. Print the value stored in the variable.

## 🧾 Program
~~~

men_stepped_on_the_moon = input("Enter a string: ")


print(men_stepped_on_the_moon)
~~~
## Output
Enter a string: Neil Armstrong
Neil Armstrong
## Result
he program successfully demonstrates how to read and print string data in Python.

The variable men_stepped_on_the_moon serves as a container in memory for the text entered.

The input() function captures the keystrokes until the "Enter" key is pressed.
